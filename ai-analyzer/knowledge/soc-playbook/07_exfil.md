# SOC 处置手册: 07_exfil (数据泄露与外传)

## 适用告警

异常上传、数据外传、大流量出站、DNS 隧道外带
MITRE T1041

## 分析要点

### 判断是否误报
- **正常业务上传**：业务系统的文件上传/备份上传到云存储
- **CDN 回源**：CDN 节点到源站的正常回源流量
- **日志上报**：Beats/Filebeat 到 ES 的正常数据上报
- **云同步**：OneDrive/坚果云等同步工具

### 判断攻击是否成功
- **流量突增**：单主机出站流量较历史基线突增数倍
- **异常目的地**：出站到未知 IP/域名（非云服务、非 CDN）
- **数据分块**：大量大小相似的出站请求（数据被分块传输）
- **DNS 外带**：DNS 查询量异常，子域名含 Base64 编码特征
- **加密外传**：HTTPS 出站流量异常大且目标为新注册域名

### 关联分析
- **出站流量统计**：告警源 IP 在告警前后的出站流量变化趋势
- **目的 IP 归属**：外传目标 IP 的地理位置、ASN、是否为 IDC/云服务
- **前置告警**：该主机是否曾触发 RCE/暴破/C2 告警（攻击链末端）
- **DNS 查询模式**：同源 IP 的 DNS 查询频率和子域名长度

## 处置流程

1. **紧急**：确认数据外传 → 立即隔离主机、评估泄露数据范围、启动应急响应
2. **高**：可疑大流量外传 → 隔离主机、封禁目标 IP/域名、深入分析
3. **中**：异常出站流量 → 监控、确认是否为正常业务
4. **低**：误报（云同步/CDN） → 白名单
