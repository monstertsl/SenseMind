# SOC 处置手册: 01_web_attack (Web应用攻击)

## 适用告警

- SQL 注入、XSS、目录遍历、文件包含、RCE、XXE、SSTI 等
- MITRE T1190

## 分析要点

### 判断是否误报
- 检查 payload 是否为扫描器特征（如 sqlmap、nuclei 的 User-Agent）
- 检查请求是否针对真实存在的接口
- 检查是否为安全扫描授权测试

### 判断攻击是否成功
- HTTP 响应状态码 200 且响应体包含敏感数据 → 可能成功
- HTTP 响应 403/404/500 → 可能失败
- 检查后续是否有 webshell 访问、异常文件创建

### 关联分析
- 同一 community_id 下的完整会话日志
- 同一源 IP 是否有后续利用行为
- 同一源 IP 的历史告警记录

## 处置流程

1. **紧急**：如确认 RCE 成功，立即隔离目标主机
2. **高**：如确认 SQL 注入成功，检查数据泄露
3. **中**：如为扫描尝试未成功，封禁 IP + 更新 WAF 规则
4. **低**：如为误报，加入白名单

## 常见误报场景
- 安全扫描器授权测试
- WAF 自身探测
- 正常业务中的特殊字符请求
