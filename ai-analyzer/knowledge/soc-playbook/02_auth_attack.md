# SOC 处置手册: 02_auth_attack (身份认证攻击)

## 适用告警

- SSH/RDP/FTP 暴力破解
- 密码喷洒、凭据填充
- 账户枚举
- MITRE T1110

## 分析要点

### 判断是否误报
- 检查是否为运维批量操作（配置管理工具）
- 检查源 IP 是否为内网可信 IP

### 判断攻击是否成功
- 登录失败 → 成功的转换模式
- 同一 IP 在短时间内尝试多个账户
- 检查目标系统日志确认是否有成功登录

### 关联分析
- 同一源 IP 的历史登录记录
- 成功登录后的异常行为（如创建新用户、修改配置）

## 处置流程

1. **紧急**：确认成功登录 → 立即禁用账户、隔离主机
2. **高**：大量失败尝试 → 封禁 IP、启用账户锁定
3. **中**：少量尝试 → 监控、加强密码策略
4. **低**：误报 → 白名单
