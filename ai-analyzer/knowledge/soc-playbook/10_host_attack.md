# SOC 处置手册: 10_host_attack (主机攻击)

## 适用告警

提权、凭据窃取、Mimikatz、LSASS 注入、计划任务持久化、SSH 密钥窃取
MITRE T1055

## 分析要点

### 判断是否误报
- **运维操作**：管理员使用 PsExec/WMI 远程管理
- **安全工具**：EDR/AV 的正常凭据扫描行为
- **域控行为**：AD 域控的正常 Kerberos 票据分发
- **监控代理**：监控系统的远程查询

### 判断攻击是否成功
- **凭据窃取**：内网出现异常 Kerberos 请求（黄金票据/白银票据）
- **LSASS 访问**：非系统进程访问 LSASS 进程（网络表现为异常认证流量）
- **持久化**：目标主机出现定时出站连接（计划任务触发的 C2 心跳）
- **权限提升**：低权限账户突然访问高权限资源

### 关联分析
- **内网认证流量**：Kerberos(88)、SMB(445)、LDAP(389) 端口异常
- **时间线**：凭据窃取 → 横向移动时间间隔通常在分钟级
- **前置告警**：主机攻击通常发生在初始入侵成功后，追溯 RCE/暴破告警
- **出站连接**：凭据窃取后是否出现 C2 回连

## 处置流程

1. **紧急**：确认凭据泄露/提权 → 立即隔离主机、重置所有相关凭据
2. **高**：可疑凭据窃取 → 隔离主机、强制 Kerberos 票据重置
3. **中**：异常认证流量 → 监控、确认是否为运维操作
4. **低**：误报（运维/域控） → 白名单
