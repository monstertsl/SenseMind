# SOC 处置手册: 05_c2 (恶意通信与C2)

## 适用告警

- 木马、C2 Beacon、反向 Shell
- Cobalt Strike、Metasploit 通信
- DGA 域名、域前置
- MITRE T1071

## 分析要点

### 判断是否误报
- 检查目标域名/IP 是否为已知安全服务
- 检查是否为合法的远程管理工具

### 判断威胁程度
- 已知 C2 IP/域名 → 确认威胁
- 未知可疑域名 + 心跳式通信 → 高度可疑
- DGA 域名特征 → 确认威胁

### 关联分析
- 同一主机的其他异常连接
- 数据外泄流量（大流量出站）
- 同一 community_id 的完整通信会话

## 处置流程

1. **紧急**：确认 C2 通信 → 立即隔离主机、启动应急响应
2. **高**：可疑 C2 通信 → 隔离主机、深入分析
3. **中**：可疑域名 → 封禁域名、监控
4. 检查是否有数据泄露、横向移动
