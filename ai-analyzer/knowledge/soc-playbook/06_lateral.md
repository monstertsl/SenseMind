# SOC 处置手册: 06_lateral (横向移动)

## 适用告警

PsExec、Pass-the-Hash、Pass-the-Ticket、SSH 跳转、RDP 内网连接、WMI 远程执行
MITRE T1021

## 分析要点

### 判断是否误报
- **运维管理**：堡垒机通过 SSH/RDP 管理内网主机
- **域控同步**：AD 域控之间的正常 SMB 复制
- **文件共享**：内网文件服务器的正常 SMB 访问
- **批量部署**：配置管理工具的远程执行

### 判断攻击是否成功
- **异常内网连接**：非运维网段主机发起 SMB/RDP/SSH 到其他内网主机
- **凭据复用**：同一账户短时间内登录多台主机
- **Pass-the-Hash**：NTLM 认证但无对应密码（凭据为哈希）
- **远程执行成功**：目标主机出现新进程、计划任务、服务

### 关联分析
- **内网→内网流量**：重点关注非运维 IP 发起的内网连接
- **攻击链时间线**：初始入侵 → 凭据窃取 → 横向移动的时间间隔
- **同源 IP 行为**：横向移动通常伴随多个目标的连接尝试
- **凭据来源**：追溯攻击者获取凭据的途径（暴破/Mimikatz/配置文件）

## 处置流程

1. **紧急**：确认横向移动成功 → 隔离源主机和目标主机、全量排查感染范围
2. **高**：可疑横向移动 → 隔离源主机、重置相关凭据
3. **中**：内网异常连接 → 监控、确认是否为运维操作
4. **低**：误报（运维/域控） → 白名单
