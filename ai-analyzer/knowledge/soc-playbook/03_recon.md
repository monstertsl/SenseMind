# SOC 处置手册: 03_recon (扫描探测行为)

## 适用告警

- 端口扫描、目录扫描、服务发现
- nmap、masscan、gobuster、nikto、nuclei 等工具特征
- MITRE T1046

## 分析要点

### 判断是否误报
- 检查是否为授权的安全评估
- 检查是否为运维监控工具
- 检查源 IP 是否为内网资产

### 判断威胁程度
- 全端口扫描 → 高威胁，攻击者在做全面侦察
- 特定端口扫描 → 中威胁，针对性探测
- 目录扫描 → 中威胁，寻找 Web 入口

### 关联分析
- 扫描后是否有漏洞利用行为（扫描 → 利用时间关联）
- 同一源 IP 的历史告警

## 处置流程

1. **高**：外网全端口扫描 → 封禁 IP、检查是否有后续利用
2. **中**：目录/服务扫描 → 封禁 IP、更新 WAF 规则
3. **低**：内网扫描 → 确认是否为授权操作
