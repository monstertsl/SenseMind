# T1071 - Application Layer Protocol

## MITRE ATT&CK 技术说明

**战术**: Command and Control (命令与控制)
**技术**: 应用层协议通信

## 描述

攻击者使用现有应用层协议（HTTP/HTTPS/DNS等）与 C2 服务器通信，以混入正常流量。

常见手法：
- HTTP/HTTPS C2 Beacon
- DNS 隧道
- 域前置 (Domain Fronting)
- DGA 域名通信

## 检测建议

- 关注 Suricata 中 C2/Trojan 类告警
- 检查异常的 DNS 查询模式（DGA、高频查询）
- 分析 TLS SNI 与证书不匹配的情况
- 关注心跳式流量（固定间隔的请求）

## 处置建议

1. 确认 C2 通信是否成功建立
2. 立即隔离受感染主机
3. 检查是否有数据外泄
4. 封禁 C2 服务器域名和 IP
5. 在防火墙/DNS 层面阻断 C2 通信
