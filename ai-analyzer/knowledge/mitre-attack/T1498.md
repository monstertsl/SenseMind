# T1498 - Network Denial of Service

## MITRE ATT&CK 技术说明

**战术**: Impact (影响)
**技术**: 网络拒绝服务

## 描述

攻击者通过耗尽目标网络或应用资源，导致合法用户无法访问服务。

## 攻击类型与特征

| 类型 | 流量特征 | 检测要点 |
|------|---------|---------|
| SYN Flood | 大量 SYN 包无 ACK 完成 | 同源 IP 大量 SYN，TCP 半连接数高 |
| UDP Flood | 大量 UDP 包到随机端口 | 同源 IP 高频 UDP，目标端口分散 |
| HTTP Flood | 大量 HTTP 请求，GET/POST | 同源 IP 高频 HTTP 请求，请求模式相似 |
| Slowloris | 慢速发送 HTTP 头保持连接 | 长时间保持的 HTTP 连接，不完整请求 |
| CC 攻击 | 针对特定 URL 的高频请求 | 单一 URL 请求量异常 |
| 反射放大 | 源 IP 为 DNS/NTP 等服务 | 源端口 53/123，响应远大于请求 |

## 成功判定

- **服务不可用**：目标端口连接数突增，正常请求超时率升高
- **带宽耗尽**：目标 IP 入站流量较基线突增数倍
- **应用层过载**：HTTP 503/502 错误率升高，响应时间变长

## 关联分析重点

- **流量突增时间点**：与告警时间戳对比，确认是否为攻击流量
- **源 IP 分布**：DDoS 通常多源 IP，单源可能是 DoS 或测试
- **目的端口/路径**：判断攻击目标是网络层还是应用层
