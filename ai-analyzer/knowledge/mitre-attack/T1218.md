# T1218 - System Binary Proxy Execution

## MITRE ATT&CK 技术说明

**战术**: Defense Evasion (防御规避)
**技术**: 系统二进制代理执行 (LOLBin)

## 描述

攻击者利用系统自带的合法二进制程序执行恶意操作，绕过应用白名单和安全检测。

## 常见 LOLBin 与特征

| 工具 | 正常用途 | 滥用方式 | 网络特征 |
|------|---------|---------|---------|
| certutil | 证书管理 | `-urlcache -split -f http://...` 下载文件 | HTTP 下载到临时目录 |
| bitsadmin | 后台传输 | `/transfer` 下载恶意文件 | HTTP 下载任务 |
| mshta | HTA 应用执行 | 执行远程 HTA 文件 | HTTP 获取 .hta 文件 |
| rundll32 | DLL 加载 | 加载远程 DLL | SMB/WebDAV 访问 |
| regsvr32 | COM 注册 | `/s /u /i:http://...` 执行远程脚本 | HTTP 下载 .sct 文件 |
| wmic | WMI 管理 | `/node:` 远程执行命令 | DCOM/WMI 协议流量 |
| msiexec | MSI 安装 | 安装远程 MSI 包 | HTTP/SMB 下载 .msi |

## 成功判定

- **下载成功**：目标主机通过 LOLBin 下载文件后出现新的进程或网络连接
- **远程执行**：wmic 或 PsExec 在内网主机间传播
- **持久化**：LOLBin 被加入计划任务或注册表自启动项

## 关联分析重点

- **出站下载**：目标主机通过非浏览器进程下载文件
- **文件类型**：下载的文件类型是否异常（.hta、.sct、.dll、.msi）
- **进程链**：LOLBin 通常作为中间跳板，需追溯父进程和子进程
