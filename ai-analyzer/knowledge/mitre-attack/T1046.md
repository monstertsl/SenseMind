# T1046 - Network Service Discovery

## MITRE ATT&CK 技术说明

**战术**: Discovery (发现)
**技术**: 网络服务扫描

## 描述

攻击者扫描网络以发现可用的服务、开放端口和运行中的应用程序，为后续攻击做准备。

常见工具：nmap、masscan、gobuster、nikto、nuclei

## 检测建议

- 监控短时间内对大量端口/IP 的连接尝试
- 关注 SYN scan 特征（大量 SYN 无后续握手）
- 目录扫描工具的特征 User-Agent

## 处置建议

1. 评估扫描范围，确认是否有敏感服务暴露
2. 检查是否有后续利用行为（扫描后紧跟漏洞利用）
3. 封禁扫描源 IP
4. 加固网络边界，关闭不必要的对外服务
