# T1068 - Exploitation for Privilege Escalation

## MITRE ATT&CK 技术说明

**战术**: Privilege Escalation (权限提升) / Exploitation
**技术**: 漏洞利用提权

## 描述

攻击者利用软件或操作系统中的漏洞提升权限，常见目标：

- 已知漏洞利用 (CVE)：Log4j、Struts2、Spring、Fastjson、Shiro、WebLogic
- 内核漏洞提权
- 服务配置错误利用

## 检测建议

- 关注包含 CVE 编号的告警
- 监控漏洞利用特征 payload（如 `${jndi:ldap://`、`%{` 等）
- 结合 community_id 关联漏洞利用前后的流量

## 处置建议

1. 确认漏洞利用是否成功（检查响应码和响应体）
2. 如成功，检查是否获得 shell 或执行了命令
3. 立即修补漏洞
4. 检查系统是否被植入后门
5. 封禁攻击源 IP
