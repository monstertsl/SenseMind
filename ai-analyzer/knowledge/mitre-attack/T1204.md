# T1204 - User Execution

## MITRE ATT&CK 技术说明

**战术**: Execution (执行)
**技术**: 用户执行恶意文件

## 描述

攻击者诱导用户执行恶意文件，或通过漏洞利用自动下载执行。在网络层面表现为恶意文件下载行为。

## 传播与下载特征

| 传播方式 | 网络特征 | 文件特征 |
|---------|---------|---------|
| 邮件附件下载 | HTTP 下载可执行文件 | .exe、.scr、.lnk、.docm |
| 恶意文档 | HTTP 下载模板或 payload | 远程模板注入 |
| 驱动下载 | 诱导访问恶意网页 | JavaScript 下载可执行文件 |
| 捆绑软件 | 下载合法软件含后门 | 正常签名但行为异常 |
| Webshell 投递 | RCE 成功后 wget/curl 下载 | 攻击者主动下载恶意文件 |

## 成功判定

- **文件下载成功**：目标主机从外部 IP 下载可执行文件（HTTP 200 + 可执行文件类型）
- **后续执行**：下载后短时间内出现出站 C2 连接
- **恶意文件特征**：文件哈希命中威胁情报、文件类型与 URL 路径不匹配

## 关联分析重点

- **下载来源**：文件下载的 URL 域名/IP 是否为已知恶意
- **下载后行为**：下载完成后目标主机是否发起新的网络连接
- **同源传播**：同一恶意文件是否在多台主机上被下载
