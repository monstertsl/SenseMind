# T1041 - Exfiltration Over C2 Channel

## MITRE ATT&CK 技术说明

**战术**: Exfiltration (数据外传)
**技术**: 通过 C2 通道外传数据

## 描述

攻击者通过已建立的 C2 通道或隐蔽通道将窃取的数据从内网传输到外部服务器。

## 外传方式与特征

| 方式 | 流量特征 | 检测要点 |
|------|---------|---------|
| HTTP POST | 大量 POST 请求，请求体异常大 | 出站 POST body 大小远超正常 |
| DNS 隧道 | 超长子域名、高频 DNS 查询 | 单个域名查询频率异常、子域名长度 >50 |
| ICMP 隧道 | 大量 ICMP 包、payload 携带数据 | ICMP 包大小异常、频率异常 |
| FTP 上传 | 出站 FTP 数据连接 | 内网主机向外部 FTP 服务器上传文件 |
| 云存储 | 大流量出站到 S3/OSS 等云存储 | 出站流量到云存储 IP 段 |

## 成功判定

- **流量异常**：单个主机出站流量较历史基线突增
- **数据分块**：大量大小相似的出站请求（数据被分块传输）
- **加密外传**：HTTPS 出站流量异常大且目标为未知域名
- **DNS 异常**：单个域名查询量远超正常，且子域名含 Base64 特征

## 关联分析重点

- **出站流量统计**：告警源 IP 在告警时间前后的出站流量变化
- **DNS 查询模式**：同源 IP 的 DNS 查询频率和子域名长度分布
- **目的 IP 归属**：外传目标 IP 的地理位置、ASN、是否为云服务
