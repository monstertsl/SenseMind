# T1190 - Exploit Public-Facing Application

## MITRE ATT&CK 技术说明

**战术**: Initial Access (初始访问)
**技术**: 利用面向公众的应用程序

## 描述

攻击者利用面向互联网的 Web 应用程序中的漏洞来获取初始访问权限。常见的攻击手法包括：

- SQL 注入 (SQLi)
- 跨站脚本 (XSS)
- 目录遍历 / 路径穿越 (Path Traversal)
- 文件包含 (LFI/RFI)
- 远程代码执行 (RCE)
- 服务端模板注入 (SSTI)
- XML 外部实体注入 (XXE)

## 检测建议

- 监控 Web 服务器日志中的异常请求（如 `../`、`%2e%2e`、`etc/passwd` 等路径遍历特征）
- 关注 Suricata 规则中的 Web 攻击类告警
- 结合 community_id 关联同一会话的多个请求，判断是否为批量扫描

## 处置建议

1. 确认攻击是否成功（检查响应状态码、响应体大小）
2. 如成功，检查 Web 服务器是否被植入 webshell
3. 修补漏洞，更新 WAF 规则
4. 封禁攻击源 IP
