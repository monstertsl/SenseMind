# T1021 - Remote Services

## MITRE ATT&CK 技术说明

**战术**: Lateral Movement (横向移动)
**技术**: 远程服务利用

## 描述

攻击者利用合法的远程服务协议在内网中横向移动，从一台受控主机跳转到另一台主机。

## 攻击手法与特征

| 协议 | 攻击手法 | 网络特征 |
|------|---------|---------|
| SMB | PsExec、WMI 远程执行、共享文件传输 | 445 端口，大量 `\\IPC$`、`\\ADMIN$` 共享访问 |
| RDP | 远程桌面登录、RDP 劫持 | 3389 端口，内网主机间 RDP 连接 |
| WinRM | PowerShell 远程执行 | 5985/5986 端口，HTTP 协议承载 |
| SSH | 横向跳转、SSH 隧道 | 22 端口，内网主机间 SSH 登录 |
| WMI | `wmic /node:` 远程执行 | 135 端口 + 动态端口 |

## 成功判定

- **Pass-the-Hash**：同一源 IP 短时间内用不同账户登录多台主机
- **PsExec 执行**：目标主机出现 `PSEXESVC.exe` 进程或 445 端口异常共享访问
- **凭据复用**：攻击者使用窃取的凭据成功登录新主机

## 关联分析重点

- **内网→内网流量**：重点关注 10.x/172.16-31.x/192.168.x 之间的连接
- **时间线**：横向移动通常发生在初始入侵成功后数小时内
- **凭据来源**：追溯攻击者获取凭据的途径（暴力破解、Mimikatz、配置文件泄露）
