# T1059 - Command and Scripting Interpreter

## MITRE ATT&CK 技术说明

**战术**: Execution (执行)
**技术**: 命令和脚本解释器

## 描述

攻击者利用命令行解释器或脚本引擎执行恶意命令，常见于 RCE 成功后的后续利用。

## 命令特征

| 解释器 | 典型命令模式 | 出现场景 |
|-------|------------|---------|
| PowerShell | `powershell -enc`、`IEX(New-Object Net.WebClient).DownloadString` | Windows RCE 后执行 |
| cmd.exe | `cmd /c`、`whoami`、`ipconfig` | Windows 命令注入 |
| Bash | `;cat /etc/passwd`、`|id`、`$(whoami)` | Linux 命令注入 |
| Python | `python -c "import os;os.system('...')"` | 跨平台 RCE |
| WMI | `wmic process call create` | Windows 远程执行 |

## 成功判定

- **命令执行成功**：HTTP 响应中包含命令输出（如 `uid=0(root)`、`whoami` 结果）
- **Webshell 访问**：攻击者通过 webshell 执行命令，URL 路径异常
- **反弹 Shell**：目标主机向外部 IP 发起反向连接（常见端口 4444、8080）
- **文件下载**：命令执行后出现 `wget`/`curl` 下载行为

## 关联分析重点

- **同 community_id**：查看命令注入的请求和响应
- **同源 IP 后续行为**：命令执行成功后通常紧跟文件下载或反弹连接
- **同目的 IP 出站**：目标主机是否在告警后发起新的出站连接
